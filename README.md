## 5. Limitations & Scientific Friction

### 5.1 Single Evaluator Bias
This audit was conducted by a single evaluator. Per Nickerson (1998), single-evaluator studies carry a 20‚Äì40% increased error rate due to confirmation bias.

**Recommended mitigation:** Apply Cohen's Kappa (Œ∫) across ‚â•2 independent evaluators.

$$\kappa = \frac{P_o - P_e}{1 - P_e}$$

- P_o = observed agreement rate  
- P_e = expected agreement rate (by chance)  
- Interpretation (Landis & Koch, 1977): Œ∫ > 0.8 = Almost Perfect, Œ∫ > 0.6 = Substantial, Œ∫ > 0.4 = Moderate

**Practical note:** Recruiting multiple independent evaluators is resource-intensive at this project scale (feasibility estimated 30‚Äì50%). Until multi-evaluator validation is complete, results should be treated as **preliminary**.

### 5.2 Baseline Definition
The 90%+ baseline is an estimate derived from published Grok benchmarks, not measured directly in this study. A controlled baseline measurement under identical conditions would strengthen the comparison.

### 5.3 Scoring Subjectivity
The Score = 0.5 category is inherently subjective. Explicit rubrics per category (logical puzzles vs. ethical dilemmas) would reduce inter-session variance.
